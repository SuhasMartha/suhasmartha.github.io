## 📦 Getting Started

1. **Clone & Install**
   ```bash
   git clone https://github.com/SuhasMartha/suhasmartha.github.io
   cd portfolio
   npm install
   ```

2. **Development**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173/](http://localhost:5173/)

3. **Build & Deploy**
   ```bash
   npm run build
   npm run deploy
   ```

---

## 📝 Usage

```bash
npm run dev      # Start development server

npm run lint     # Lint codebase
```
